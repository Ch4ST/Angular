Pourquoi utiliser Angular ?

Angular est un framework est un outil très performant pour la création d'application web.
Notamment le fait qu'on puisse l'utiliser avec plusieurs langage de code comme :

  • Javascript
  • Dart
  • Typescript

    • Il est également possible d'utiliser Angular quelque soit sont architecture serveur.
    La remaniabilité du code source est également l'un des grand avantage d'Angular.
    La communauté d'Angular est très grande cela permet d'avoir de l'aide sur internet ou sur des forums en cas de blocage.
